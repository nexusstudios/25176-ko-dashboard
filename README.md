# Knockout Build Dashboard

## Purpose of this repo
This repository generates a static build dashboard HTML page from tracked GitHub workflow data for Knockout projects.

## Derives from
This repository is derived from the open-source `lincc-frameworks/lf-workflow-dash` project and adapted for the Knockout build monitoring use case.

## Infrastructure
- Dashboard is generated by a GitHub Action workflow that runs on a half an hour schedule.
- Dashboard has been customised to show build data for Knockout projects, and to include a login page for access control.
- Dashboard generated files are synced to an S3 bucket.
- CloudFront serves the dashboard frontend.
- A Lambda function enforces username/password access before users can view the site.

## Notes
The dashboard will refresh automatically, with CloudFront set up to serve the latest updated version of the dashboard. This has been tested, and no matter how long you leave the page open, it will refresh. 
If you are not seeing the latest data, please try refreshing your browser cache or opening the site in an incognito window.
